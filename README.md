# Sisterhood Information & Support System

SISS! (Sisterhood Information & Support System) is a web-based platform designed to support women's reproductive health education through menstrual cycle tracking, personalized health insights, and interactive learning resources. The application was developed with the goal of helping users better understand the biological processes occurring throughout their menstrual cycle in a simple, accessible, and engaging way.

Through SISS!, users can record their menstrual cycles, monitor mood and symptoms, view cycle predictions, and explore educational content about reproductive anatomy, hormonal changes, and the different phases of the menstrual cycle. By combining health education with digital technology, the platform aims to improve reproductive health literacy while encouraging users to become more aware of patterns within their own bodies.

The application includes cycle tracking features, an interactive menstrual calendar, mood and symptom monitoring, educational modules on reproductive health, hormone explanations, anatomy visualizations, and automatically generated insights based on recorded data. In addition, SISS! is built as a Progressive Web App (PWA), allowing users to install it on their devices and access core features even when internet connectivity is limited.

This project was developed using HTML5, CSS3, JavaScript (ES Modules), LocalStorage, Chart.js, Service Workers, and Web App Manifest technologies. The current version focuses on client-side functionality and privacy-friendly local data storage, while future development plans include cloud synchronization, user authentication, advanced health analytics, and personalized recommendations.

SISS! reflects the intersection of information technology and health education, demonstrating how digital solutions can be used to make scientific knowledge more approachable and meaningful in everyday life.
